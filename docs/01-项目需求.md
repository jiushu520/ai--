# GoldPilot 项目需求文档

## 一、产品概述

### 1.1 产品定位
一个面向黄金交易客户的**实时交易信号跟单展示系统**，通过可视化信号告诉客户什么时候买、什么时候卖，并实时展示胜率和盈利数据。

### 1.2 解决的问题
| 问题 | 解决方案 |
|------|----------|
| 客户不懂技术分析 | 用可视化信号告诉客户什么时候买、什么时候卖 |
| 不知道跟单效果 | 实时展示当天的胜率、盈利数据，建立信任 |
| 感觉不够专业 | 用AI概念包装，提升产品价值感 |

### 1.3 目标用户
- 黄金交易客户
- 需要跟单建议的投资者
- 金融服务团队

## 二、功能需求

### 2.1 核心功能

#### 功能1：智能交易信号提醒
**用途**：告诉客户当前应该做多还是做空

**技术实现**：
```javascript
// 趋势判断
趋势判断：EMA89 < EMA233 = 空头，EMA89 > EMA233 = 多头

// 信号触发
做空信号：空头趋势 + EMA21下穿EMA89（死叉）
做多信号：多头趋势 + EMA21上穿EMA89（金叉）

// 止盈目标
止盈价格 = 当前周期ATR14 × 5
```

**展示方式**：
- 在K线图上标记信号点
- 显示"做多↑"、"做空↓"、"止盈●"
- 支持多周期（1分钟、5分钟、15分钟等）

#### 功能2：跟单效果展示
**用途**：让客户看到信号有多准

**展示内容**：
```
┌─────────────────────────────────────┐
│ 今日信号表现                         │
├─────────────────────────────────────┤
│ 信号数:  8次                         │
│ 胜率:    75%                         │
│ 盈利:    +$1,850                     │
│ 亏损:    -$450                       │
│ 净盈利:  +$1,400                     │
└─────────────────────────────────────┘
```

#### 功能3：AI分析建议
**用途**：给客户专业的交易建议

**展示内容**：
- 当前市场状态分析
- AI趋势预测
- 具体操作建议
- 风险提示

#### 功能4：实时行情展示
- 现货黄金实时价格
- 涨跌幅计算
- K线图表（支持多周期切换）
- 支撑压力位标注

#### 功能5：MT4/MT5账户对接
**测试账号信息**：
```
账号：27238218
服务器：VTMarkets-Live 8
观摩密码：Abc1234@
```

**展示数据**：
- 账户净值
- 可用保证金
- 持仓手数
- 今日盈亏
- 风险占用

## 六、技术选型

### 6.1 前端技术栈
- **框架**：React 18 + TypeScript
- **构建工具**：Vite
- **样式**：TailwindCSS
- **K线图**：Lightweight Charts
- **状态管理**：React Context / Zustand
- **HTTP客户端**：Axios
- **WebSocket**：Socket.IO Client

### 6.2 后端技术栈
- **运行环境**：Node.js 20 LTS
- **Web框架**：Express.js
- **WebSocket**：Socket.IO
- **数据库**：MongoDB + Mongoose
- **MT4/MT5对接**：MetaAPI SDK
- **进程管理**：PM2
