## 快速部署步骤

### 方法一：使用SSH手动部署（推荐）

1. **打开 PowerShell 或 CMD**

2. **连接到服务器**
   ```bash
   ssh root@43.138.229.56
   # 输入密码
   ```

3. **执行部署命令**
   ```bash
   cd /var/www/goldpilot
   git pull
   cd goldpilot-frontend
   npm run build
   pm2 restart all
   nginx -s reload
   ```

4. **访问网站**
   打开浏览器访问: http://43.138.229.56

### 方法二：一键部署脚本

在本地 PowerShell 执行：
```powershell
# 设置服务器密码
$PASSWORD = "你的服务器密码"

# 使用echo传递密码
echo $PASSWORD | ssh root@43.138.229.56 "cd /var/www/goldpilot && git pull && cd goldpilot-frontend && npm run build && pm2 restart all && nginx -s reload"
```

### 方法三：配置SSH密钥（一次性配置）

1. **生成SSH密钥**（如果还没有）
   ```bash
   ssh-keygen -t rsa -b 4096
   # 一路回车
   ```

2. **复制公钥到服务器**
   ```bash
   type C:\Users\你的用户名\.ssh\id_rsa.pub | ssh root@43.138.229.56 "mkdir -p ~/.ssh && cat >> ~/.ssh/authorized_keys"
   ```

3. **之后就可以免密登录部署**
   ```bash
   ssh root@43.138.229.56 "cd /var/www/goldpilot && git pull && cd goldpilot-frontend && npm run build && pm2 restart all"
   ```

## 常见问题

**Q: SSH连接被拒绝**
A: 检查防火墙设置，确保SSH端口22开放

**Q: npm install失败**
A: 运行 `cd /var/www/goldpilot/goldpilot-frontend && npm cache clean --force`

**Q: 网页无法访问**
A: 检查Nginx状态 `sudo systemctl status nginx`

**Q: 后端无响应**
A: 检查PM2状态 `pm2 list`，必要时 `pm2 restart all`
