## 2026-05-16 - 前端React项目搭建（续）

### ✅ 今日完成
1. **创建Vite + React + TypeScript项目**
   - 使用 `npm create vite@latest` 创建项目
   - 模板：react-ts
   - 项目位置：`goldpilot-frontend/`

2. **安装项目依赖**
   - 运行时依赖：
     - `lightweight-charts` - K线图库
     - `socket.io-client` - WebSocket客户端
     - `axios` - HTTP客户端
     - `zustand` - 状态管理
     - `antd` - Ant Design组件库
     - `dayjs` - 日期处理
   - 开发依赖：
     - `tailwindcss` - CSS框架
     - `postcss` - CSS处理
     - `autoprefixer` - CSS自动前缀
     - `@types/node` - Node类型定义

3. **配置项目**
   - 配置TailwindCSS（`tailwind.config.js`）
   - 配置PostCSS（`postcss.config.js`）
   - 配置路径别名（`vite.config.ts`、`tsconfig.app.json`）
   - 配置环境变量（`.env.example`、`.env.development`）

4. **创建目录结构**
   ```
   src/
   ├── components/          # 组件目录
   │   ├── Header/
   │   ├── PriceCard/
   │   ├── Chart/
   │   ├── SignalPanel/
   │   ├── AccountCard/
   │   ├── EventList/
   │   └── DecisionCard/
   ├── hooks/              # 自定义Hooks
   ├── services/           # API服务
   ├── utils/              # 工具函数
   ├── types/              # 类型定义
   └── context/            # Context
   ```

5. **创建基础文件**
   - `src/types/index.ts` - 类型定义（Candle、Signal、Account等）
   - `src/utils/format.ts` - 格式化工具函数
   - `src/utils/constants.ts` - 常量配置
   - `src/services/api.ts` - API服务封装
   - `src/services/websocket.ts` - WebSocket服务封装
   - `src/index.css` - 全局样式（TailwindCSS + Ant Design）
   - `src/App.tsx` - 根组件（测试页面）

6. **测试验证**
   - 开发服务器启动成功（`npm run dev`）
   - 访问 http://localhost:5173 正常显示
   - 无编译错误

### 🚧 遇到的问题
- **问题1**：`npx tailwindcss init -p` 报错
  - **解决**：直接创建 `tailwind.config.js` 和 `postcss.config.js` 文件

### 💡 解决方案
无重大问题，项目初始化顺利。
